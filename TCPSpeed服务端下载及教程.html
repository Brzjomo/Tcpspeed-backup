TCPSpeed服务端下载及教程

TCPSpeed服务端,支持Linux,Windows.

当前版本3.2

----------------------------------------------------------------------------------------------------------------------------------------------------------

1.Linux系统

一键安装命令
rm -f install_tcpspeed.sh ; wget http://www.tcpspeed.co/tcpspeed/install_tcpspeed.sh ; chmod +x install_tcpspeed.sh ; ./install_tcpspeed.sh 2>&1 | tee install.log

如果提示错误 wget: command not found,请手动安装wget,安装命令 yum -y install wget 或 apt-get install wget

初次安装建议先不要修改任何设置,以免错误配置导致无法连接!

debian,ubuntu下如果执行脚本出错,请切换到dash,
切换方法: sudo dpkg-reconfigure dash 选no


激活命令
sh /xs/dactivate.sh 激活码

激活命令2 (2016.9.17日前安装)
sh /xs/activate.sh

检查激活状态
sh /xs/check_activation.sh
启动
sh /xs/start.sh

重新启动
sh /xs/restart.sh
停止
sh /xs/stop.sh

日志
tail -f /xs/server.log
更新
执行一键安装会自动完成更新.

卸载
sh /xs/uninstall.sh


修改监听端口 (默认udp 130和tcp 130 ,由于tcpspeed的工作原理,请不要在本机防火墙开放tcpspeed监听的tcp端口)
mkdir -p /xs/cnf/ ; echo 端口号 > /xs/cnf/listen_port ; sh /xs/restart.sh
假设服务器地址为1.2.3.4,端口为8888,客户端登录地址需改为 1.2.3.4:8888


设置密码,默认无密码,可添加多个密码
mkdir -p /xs/cnf/ ; echo 密码 >> /xs/cnf/password ;sh /xs/restart.sh

删除,修改密码
编辑密码文件/xs/cnf/password,
一行代表一个密码,删除或修改相应行.
vi /xs/cnf/password
修改后重启服务端生效
sh /xs/restart.sh

查看密码
cat /xs/cnf/password

取消密码
rm -rf /xs/cnf/password ; sh /xs/restart.sh

每天晚上3点自动重启
crontab -e
加入
0 3 * * * sh /xs/restart.sh

详细自动重启方法
注:仅适合vi/vim编辑器,nano编辑器按提示操作即可
运行crontab -e
按i切换到插入状态
输入 0 3 * * * sh /xs/restart.sh
输入完成后按ESC键,再按组合键Shift和:(分号)
输入wq回车保存完成

----------------------------------------------------------------------------------------------------------------------------------------------------------

2.Windows系统

下载地址: tcpspeed_server.zip

系统需安装java运行环境,以及winpcap.

安装:下载解压.

运行:执行start_windows.bat.

设置密码:

cnf目录下新建文件,文件名为password (无后缀) , 文件内一行代表一个密码.

设置端口:

cnf目录下新建文件,文件名为listen_port (无后缀) , 文件内容为端口号.

激活:
在服务器上用浏览器打开
http://d.tcpspeed.co:8080/xs/active.jsp?type=submit_code&active_code=你的激活码

检查激活:
在服务器上用浏览器打开
http://d.tcpspeed.co:8080/xs/active.jsp?type=check_active_text
